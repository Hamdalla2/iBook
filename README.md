# iBook #
- Mini-project for front-end interview task at xngage.

## How to run ##
- Install dependencies: ```npm i```
- Run the server & databse: ```npm run server```
- Run the app: ```npm start```
 
## Creation ##
- front end: react ts
- back end: node js express
- database: mongodb atlas
